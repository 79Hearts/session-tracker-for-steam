# Session Tracker for Steam — Future Changes

Running list of improvements to batch up later.

- [ ] **Longest-session leaderboard** — surface your top 10 single sessions with date / game / duration.
- [ ] **First-played / last-played markers per game** — show "First played: April 12" and "Last played: 2 days ago" on the summary cards.
- [ ] **Weekly comparison overlay** — on the timeline bar, ghost the same weekday from the previous week behind today's session segments.
- [ ] **CSV export of session data** — let users dump their full local history.
- [ ] **Custom retention setting** — allow 7 / 30 / 90 / 365 days, default 30.
